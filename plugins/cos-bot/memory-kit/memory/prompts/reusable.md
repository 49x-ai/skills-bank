# Reusable Prompts

Prompt patterns that are useful across projects.

## Update memory

```text
Update memory based on the durable decisions, constraints, and reusable
learnings from this session. Do not save raw transcript details.
```

## Compact inbox

```text
Compact memory/inbox into the relevant durable memory files. Keep only
reusable context; delete or archive noise.
```

## Reorient on a project

```text
Read PROTOCOL.md and memory/MEMORY.md, then memory/active.md and the
relevant file under memory/projects/. Summarize where things stand and
what the open loops are.
```
